# Credit Card Fraud Detection using ANN

## Project Overview
This project detects fraudulent credit card transactions using an Artificial Neural Network (ANN).

## Dataset
Credit Card Fraud Detection Dataset

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- TensorFlow / Keras
- Matplotlib
- Seaborn
- SMOTE

## ANN Architecture
- Input Layer
- Dense Layer (32 neurons, ReLU)
- Dense Layer (16 neurons, ReLU)
- Output Layer (1 neuron, Sigmoid)

## Performance Metrics
- Accuracy
- Precision
- Recall
- F1-Score
- Confusion Matrix

## Handling Imbalanced Data
SMOTE (Synthetic Minority Oversampling Technique) was used to balance fraudulent and genuine transactions.

- fraud_model.h5
